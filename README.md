# 🖥️ Python Device Health Monitor

A system automation tool built with Python to monitor device performance and perform system operations with real-time notifications.

---

## 🚀 Features

- ✅ Monitor CPU usage
- ✅ Monitor RAM usage
- ✅ Process monitoring
- ✅ Restart system functionality
- ✅ Desktop notifications
- ✅ Automatic health checks
- ✅ CLI + GUI Interface

---

## 🛠️ Tech Stack

- Python
- psutil
- plyer
- customtkinter
